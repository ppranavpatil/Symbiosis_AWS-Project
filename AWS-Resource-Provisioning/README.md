# AWS Resource Provisioning using Python (Boto3)

## 📌 Project Overview

This project demonstrates how to automate AWS resource provisioning using Python and the AWS SDK (Boto3). Instead of creating AWS resources manually through the AWS Management Console, Python scripts are used to create and manage infrastructure programmatically.

---

## 🚀 AWS Services Used

- Amazon EC2
- Amazon S3
- AWS IAM
- Python 3
- Boto3
- AWS CLI

---

## ✨ Features

- Create Amazon EC2 Instance
- Create Amazon S3 Bucket
- Automate AWS resource provisioning
- Configure AWS using AWS CLI
- Learn Infrastructure Automation using Python

## ⚙️ Prerequisites

- AWS Account
- Python 3.x
- AWS CLI
- Boto3

Install Boto3

```bash
pip install boto3
```

Configure AWS CLI

```bash
aws configure
```

---

## ▶️ Run the Project

Create an S3 Bucket

```bash
python create_s3.py
```

Create an EC2 Instance

```bash
python create_ec2.py
```
