# MovieStream – AWS OTT Platform

## Project Overview

MovieStream is a static OTT-style movie streaming website hosted on Amazon EC2. Movie posters are stored in Amazon S3, and users can watch official YouTube trailers through an interactive popup player.

---

## Features

- Responsive OTT Interface
- Movie Posters stored in Amazon S3
- YouTube Trailer Popup
- Dark Netflix-style UI
- HTML + CSS + JavaScript
- Hosted on Amazon EC2

---

## AWS Services Used

- Amazon EC2
- Amazon S3
- IAM
