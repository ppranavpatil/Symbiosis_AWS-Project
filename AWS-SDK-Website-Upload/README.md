# AWS S3 Static Website Hosting

## 📌 Project Overview
This project demonstrates how to host a static website using Amazon S3. The website files are uploaded using Python (Boto3), and the S3 bucket is configured for public static website hosting.

## 🚀 AWS Services Used
- Amazon S3
- IAM
- Python
- Boto3

## 📂 Project Structure

AWS-S3-Static-Website/
│── index.html
│── style.css
│── upload.py
│── requirements.txt
│── README.md
└── Screenshots/

## Features
- Static website hosting
- Public website access
- Automated file upload using Python
- Bucket policy configuration

## Steps
1. Create S3 Bucket
2. Enable Static Website Hosting
3. Upload website files
4. Configure Bucket Policy
5. Access website endpoint
